## 1. 产品概述

冷藏集装箱温控复盘桌面工具，面向冷链企业质控专员，在运输结束后整理每票货的温控表现。通过导入设备导出的温湿度、开关门和定位记录，结合客户要求的温区与运输节点，实现可视化复盘标注与结构化报告生成，支撑冷冻肉类、疫苗原料、高价值海鲜等品类的质量追溯与理赔沟通。

- 解决问题：运输结束后温控数据分散、复盘依赖人工记忆、报告格式不统一、责任阶段难以界定
- 目标用户：冷链企业质控专员、运营主管、理赔沟通人员
- 核心价值：将温控复盘从"凭经验判断"升级为"数据驱动决策"，缩短复盘时间50%以上，提升报告一致性与可追溯性

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用场景 | 核心权限 |
|------|----------|----------|
| 质控专员 | 日常复盘标注、报告生成 | 导入数据、标注异常、生成报告 |
| 运营主管 | 审核复盘结果、提取运营指标 | 查看报告、导出汇总 |
| 理赔专员 | 对外沟通、索赔支持 | 查看报告、生成客户版摘要 |

### 2.2 功能模块

1. **导入记录窗口**：设备数据导入、运输参数录入、数据预览与校验
2. **复盘标注窗口**：时序曲线展示、异常框选标注、判定依据填写、标注汇总
3. **报告生成窗口**：模板选择、关键指标摘要、报告预览与导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 导入记录 | 数据文件导入 | 支持CSV/Excel格式的温度、湿度、开关门、定位记录批量导入，自动解析时间戳与数值 |
| 导入记录 | 运输参数录入 | 录入客户温区要求（如-18℃~-15℃）、货品类型（冷冻肉类/疫苗原料/高价值海鲜）、装卸时间、起运港/目的港等节点 |
| 导入记录 | 数据校验概览 | 导入后展示数据时间范围、采样间隔、缺失段提示、数据量统计 |
| 复盘标注 | 时序曲线面板 | 按时间顺序展示温度曲线（含温区上下限标注线）、湿度曲线、开关门事件标记、定位里程叠加 |
| 复盘标注 | 异常框选标注 | 鼠标框选波动段，弹出标注面板，选择异常类型（预冷不足/堆场断电/门封检查/长时间开门/设备误报），填写判定依据 |
| 复盘标注 | 标注汇总侧栏 | 列出所有已标注异常，显示类型、时间段、时长，支持编辑删除 |
| 报告生成 | 模板选择 | 选择客户版/内部运营版/理赔沟通版模板，不同模板侧重不同指标和措辞 |
| 报告生成 | 关键指标摘要 | 自动计算超限时长、最大偏差、异常次数、责任阶段分布 |
| 报告生成 | 报告预览与导出 | 预览结构化复盘报告（含关键节点、超限时长、责任阶段、处理建议），支持打印/导出PDF |

## 3. 核心流程

质控专员打开工具后，首先在导入记录窗口导入设备数据和录入运输参数；系统校验数据完整性后进入复盘标注窗口；专员沿时间线浏览温控曲线，对异常波动段框选并标注类型和依据；所有标注完成后进入报告生成窗口，选择合适的报告模板，系统自动汇总关键指标，专员确认后生成复盘摘要报告。

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：工业监控台风格（Industrial Control Room），深色背景配合冷色调，呈现冷链行业专业感
- **主色**：深蓝灰（#0F1923）为底色，冰蓝（#00D4FF）为主强调色，极光青（#0AFFCE）为次强调色
- **告警色**：琥珀橙（#FF8C00）为预警，冰红（#FF3366）为超限/异常
- **按钮风格**：圆角微凸（2px），深色底+冰蓝边框，hover时填充冰蓝
- **字体**：标题使用 JetBrains Mono（等宽工业感），正文使用 Noto Sans SC
- **布局风格**：左侧窄导航+主内容区，三窗口Tab切换，数据密集但层次分明
- **图标风格**：线性图标（lucide-react），1.5px线宽，与工业监控风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 导入记录 | 数据文件导入 | 拖拽上传区域（虚线边框+冰蓝高亮），文件列表表格，状态徽章（解析中/完成/错误） |
| 导入记录 | 运输参数录入 | 分组表单卡片（温区参数/货品信息/运输节点），数字输入带单位后缀，时间选择器 |
| 导入记录 | 数据校验概览 | 统计卡片（时间范围/采样数/缺失段），进度条式数据完整性指示器 |
| 复盘标注 | 时序曲线面板 | 大面积折线图（温度主线+温区带+事件标记），底部时间轴缩放条，右侧Y轴温度刻度 |
| 复盘标注 | 异常框选标注 | 曲线上可拖拽选区（半透明冰蓝填充），弹出标注浮层（类型下拉+依据文本框） |
| 复盘标注 | 标注汇总侧栏 | 右侧固定侧栏，异常列表（类型图标+时间范围+时长），编辑/删除操作 |
| 报告生成 | 模板选择 | 三列卡片（客户版/运营版/理赔版），选中态冰蓝边框+发光效果 |
| 报告生成 | 关键指标摘要 | 四宫格指标卡（超限时长/最大偏差/异常次数/责任分布），数值大字号+标签小字号 |
| 报告生成 | 报告预览与导出 | A4纸样式预览区，顶部操作栏（打印/导出PDF），结构化报告内容 |

### 4.3 响应式策略

- 桌面优先设计，最小支持1280px宽度
- 曲线图区域弹性伸缩，侧栏固定宽度320px
- 报告预览区域居中A4布局，两侧留白
